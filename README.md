## ✨ Project Overview

This project is a **web scraper for Booking.com** built using **Python** and **Selenium**.  
It opens Booking.com in a real browser, extracts hotel listing details, and saves the data into a **CSV file** for further analysis.

---

## 🎯 Features

✅ Scrapes hotel listings from Booking.com  
✅ Extracts:
- Hotel Name
- Price
- Rating
- Hotel Link

✅ Saves scraped data into a CSV file  
✅ Uses **Selenium** to handle dynamic website content  
✅ Beginner-friendly and easy to run  

---

## 🛠 Tech Stack

- **Python**
- **Selenium**
- **CSV Module**
